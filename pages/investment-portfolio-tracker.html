Investment Portfolio Tracker

An ongoing passion project, initially built on Google Sheets, with aspirations to become a standalone web-app. This investment tracker automatically tracks an individual's cryptocurrencies and NFTs in real-time, offering a comprehensive view of an individual's investments by pulling API data directly from exchanges.

Features:

  • Obtain an overall portfolio value combined from cryptocurrencies, staking, and even the controversial asset class NFTs, which are notorious for being difficult to track.
  • Directly connect the exchanges you use to the tracker via read-only API calls to automatically pull and aggregate data.
  • Track assets that are transfered to different wallets, including hardware wallets.
  • Logs data daily to create historical portfolio metrics such as total profit, assets, deposits, ROI, etc.
  • Assists in rectifying issues/mislabeled transactions identified in automated tax software.
  • Redundant price information pulled from multiple sources to improve accuracy.
  • Set up alerts for changes, price jumps, or portfolio value.
  • Risk analysis.

Logs daily portfolio metrics such as:

  • Portfolio Distribution
  • Profit Distribution
  • Return On Investment (%) Over Time
  • Total Assets, Fiat Deposits and ROI comparison
  • Avg. Sell Over Buy %

Created using automated Google Apps Scripts, custom made Google Sheets extensions, SQL queries, and a range of advanced spreadsheet functions for data aggregation.
