CORPORATE SOCIAL MEDIA WEBSITE

This project, constructed with technologies such as React, Bootstrap, Express, and GraphQL ontop of Node, provides a wealth of features including user image upload, post editing with text formatting options, post reactions, follow/unfollow functionality, and two-factor authentication (2FA). The project was tested using Jest and its documentation was generated using JDocs for comprehensive reference.

  • This project was built during my studies, achieving a high score.
  • Built for desktop view only as per requirements.
  • Hosted on AWS for demostration purposes.
